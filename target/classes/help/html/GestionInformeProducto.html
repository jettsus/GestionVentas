Gestión de informes de ventas - Ayuda

En esta ventana, puedes generar informes de las ventas listadas en la sección de "Gestión de ventas". A continuación, se detalla la funcionalidad:

  • Opciones de informe: Puedes elegir entre dos tipos de informes, seleccionando un formato para cada uno:
    • Exportar todos los productos: Genera un informe con la lista completa de ventas, incluyendo detalles como vendedor, producto, fecha, total y cantidad. Selecciona el formato (PDF o HTML).
    • Exportar productos con gráfico: Genera un informe de las ventas con un gráfico visual, además de los detalles de las ventas. Selecciona el formato (PDF o HTML).
  • Opciones de formato:
    • PDF: Si está seleccionada, el informe se generará en formato PDF.
    • HTML: Si está seleccionada, el informe se generará en formato HTML, que puede visualizarse en un navegador web.
  • Botón "Exportar": Después de seleccionar un formato para uno o ambos tipos de informe (Exportar todos los productos o Exportar productos con gráfico), debes pulsar el botón "Exportar" para generar el informe. El informe reflejará el estado actual de la lista de ventas, incluyendo los filtros aplicados en la ventana de "Gestión de ventas".

EN LA BARRA DE MENÚ (en la ventana de Gestión de informes de ventas):

  • Archivo: Al pulsarlo, se despliega un menú con la siguiente opción:
    • Volver: Regresa a la ventana de "Gestión de ventas".
  • Ayuda: Abre esta página de ayuda al pulsarla o al presionar F1.
