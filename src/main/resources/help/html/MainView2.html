Menu principal - Ayuda

Pulsando F1 en la ventana principal, se abrirá esta página de ayuda.

AQUÍ LOS BOTONES Y SU FUNCIONALIDAD:

  • Gestión de productos: Este botón permite al administrador o vendedor acceder a la sección donde se pueden gestionar los productos, incluyendo la adición, edición y eliminación de productos del sistema.
  • Gestión de ventas: Este botón dirige al administrador o vendedor a la sección dedicada a la gestión de ventas, donde se pueden registrar, consultar y analizar las ventas realizadas.
  • Gestión de usuarios: Este botón abre la sección para la administración de usuarios, permitiendo crear, modificar o eliminar cuentas de usuario, así como asignar roles y permisos.

EN LA BARRA DE MENÚ:

  • Archivo: Permite acceder a opciones básicas del sistema.
  • Ayuda: Abre esta página de ayuda al pulsarla o al presionar F1, proporcionando información sobre las funcionalidades de la interfaz.
  • Cerrar sesión: Permite al administrador o vendedor cerrar su sesión actual y salir del sistema de forma segura.
