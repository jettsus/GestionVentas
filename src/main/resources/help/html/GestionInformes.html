Informe todos los productos - Ayuda

En esta ventana, puedes generar un informe de todos los productos listados en la sección de "Gestión de productos". A continuación, se detalla la funcionalidad:

  • Opciones de formato: Puedes seleccionar uno o ambos formatos para el informe marcando las casillas correspondientes:
    • PDF: Si está seleccionada, el informe se generará en formato PDF, incluyendo la lista de productos con sus detalles (Nombre, Precio, Stock).
    • HTML: Si está seleccionada, el informe se generará en formato HTML, que puede visualizarse en un navegador web.
  • Botón "Exportar": Después de seleccionar al menos un formato (PDF, HTML o ambos), debes pulsar el botón "Exportar" para generar el informe. El informe reflejará el estado actual de la lista de productos, incluyendo los filtros aplicados en la ventana de "Gestión de productos".

EN LA BARRA DE MENÚ (en la ventana de Informe todos los productos):

  • Archivo: Al pulsarlo, se despliega un menú con las siguientes opciones:
    • Volver: Regresa a la ventana principal (Gestion de productos).
  • Ayuda: Abre esta página de ayuda al pulsarla o al presionar F1.
