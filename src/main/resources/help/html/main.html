Bienvenido a la Ayuda del Sistema de Gestión

Esta aplicación sirve para gestionar productos, ventas y usuarios, con funcionalidades para añadir, editar, eliminar y generar informes. Esta página es la ayuda principal y proporciona enlaces a las páginas de ayuda de cada ventana de la aplicación.

La aplicación consta de las siguientes ventanas, cada una con su propia página de ayuda:

  • Ventana Principal (Menú Principal): Acceso a las principales funcionalidades del sistema.
  • Gestión de Productos: Permite administrar los productos del sistema.
  • Añadir Producto: Ventana para agregar un nuevo producto.
  • Editar Producto: Ventana para modificar los detalles de un producto existente.
  • Gestión de Informes de Productos: Genera informes de los productos en diferentes formatos.
  • Gestión de Ventas: Permite administrar las ventas del sistema.
  • Añadir Venta: Ventana para registrar una nueva venta.
  • Editar Venta: Ventana para modificar los detalles de una venta existente.
  • Gestión de Informes de Ventas: Genera informes de las ventas, con opción de incluir gráficos.
  • Gestión de Usuarios: Permite administrar los usuarios del sistema.
  • Añadir Usuario: Ventana para agregar un nuevo usuario.
  • Editar Usuario: Ventana para modificar los detalles de un usuario existente.
