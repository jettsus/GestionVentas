Gestión de ventas - Ayuda

En esta ventana, puedes administrar las ventas del sistema. A continuación, se detalla la funcionalidad:

  • Tabla de ventas: Muestra una lista de ventas con su ID, vendedor, producto, fecha, total y cantidad. Puedes pulsar sobre cualquier campo de la tabla para editar el valor directamente.
  • Botón "Eliminar": Permite eliminar una venta seleccionada de la lista.
  • Filtros (Vendedor, Producto, Precio máximo): Puedes introducir valores en los campos de filtro y pulsar "Aplicar filtros" para mostrar solo las ventas que cumplan con esos criterios. También puedes aplicar los filtros presionando la tecla Enter después de ingresar los valores.
  • Botón "Aplicar filtros": Aplica los filtros introducidos para actualizar la tabla de ventas. También se activa al presionar la tecla Enter.
  • Botón "Limpiar filtros": Restablece los filtros y muestra todas las ventas nuevamente.
  • Paginación ("Anterior", "Siguiente", "Página"): Permite navegar entre páginas si hay muchas ventas en la lista.

EN LA BARRA DE MENÚ (en la ventana de Gestión de ventas):

  • Archivo: Al pulsarlo, se despliega un menú con las siguientes opciones:
    • Editar: Permite modificar los datos de una venta seleccionada en la tabla (también puedes editar directamente pulsando un campo de la tabla).
    • Añadir: Permite agregar una nueva venta al sistema.
    • Volver: Regresa a la ventana principal (Menú principal).
  • Ayuda: Abre esta página de ayuda al pulsarla o al presionar F1.
